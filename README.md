# constitute-ui

`constitute-ui` is the shared first-party browser surface layer.

It provides reusable presentation primitives, shared chrome, prepared-state
rendering patterns, runtime-surface client helpers, app contract helpers,
module-role resolution, generic platform adapters, and common operator UI
conventions for Constitution browser apps.

Shared UI components render prepared posture and narrow controls. They do not
own protocol records, route truth, service admission, projection truth, or
authority state.
